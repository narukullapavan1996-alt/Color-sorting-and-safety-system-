# 🎨 Color Sorting and Safety System 🛡️

Welcome to the **Color Sorting and Safety System**! This project is a Computer Vision (CV) application designed to automate color-based sorting and monitor safety, likely along a conveyor belt system. 

Whether you're here to check out the code for an expo or looking to contribute, this guide will help you navigate the project.

## 🌟 What Does This Project Do?

This system uses computer vision to track objects, identify their colors, and monitor their movement kinematics in real-time. It's built to ensure both operational efficiency (sorting) and environmental safety.

**Key Capabilities:**
*   **Real-time Vision Processing:** Uses camera feeds to detect and analyze objects (`vision.py`).
*   **Color Classification:** Accurately identifies and sorts items based on color profiles (`color_data.py`).
*   **Object Tracking & Telemetry:** Tracks movement and logs kinematic data to CSV files for further analysis (`tracker.py`, `tracking_data.csv`, `KINEMATIC_TELEMETRY.csv`).

## 📂 Project Structure

Here is a quick rundown of the main files in the `CV EXPO` directory and what they do:

*   **`main.py`**: The core entry point of the application. Run this to start the system.
*   **`vision.py`**: Handles the image processing and camera feed integrations.
*   **`tracker.py`**: Manages the logic for tracking objects across frames.
*   **`color_data.py`**: Contains the logic or datasets used for accurate color matching and sorting.
*   **`evidence/` & `SCREEN_CAPTURE_*.jpg`**: Contains sample captures and evidence of the system working in action.
*   **`*.csv` files**: Telemetry and tracking data outputs generated by the system.
*   **`conveyor belt.png`**: A visual asset/diagram of the physical setup.
